A set $E$ in a measurable space $(X, \mathcal{A})$ is called **measurable** if it belongs to the $\sigma$-algebra $\mathcal{A}$. The $\sigma$-algebra $\mathcal{A}$ contains sets for which a measure can be consistently defined.
